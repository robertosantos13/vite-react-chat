# :hammer: Funcionalidades do projeto
- `Funcionalidade 1`: Registrar e Logar usando Firebase

## Criar projeto
```
npm create vite@latest
```

## Abrir o pasta do projeto
```
cd nome do projeto
```

## Instalar os pacotes
```
npm install
```

### Rodar o projeto em dev
```
npm run dev
```
